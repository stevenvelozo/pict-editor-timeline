# Pict-Editor-Timeline

> Visual timeline editor for building multi-beat video storyboards in the Pict framework

- Add, reorder, duplicate, and configure cuts with prompts and durations
- Optional start/end frame image slots with a pluggable media adapter
- Live duration strip and total-duration readout
- Exports clean JSON for any downstream video-generation system
